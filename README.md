# 保研模拟器 (Baoyan Simulator)

一个模拟大学生从高考到保研全过程的互动游戏，体验保研路上的各种挑战与机遇。

## 🌟 功能特性

### 游戏流程
- **高考阶段**：模拟高考分数，影响后续大学选择
- **大学选择**：根据高考分数选择合适的大学
- **专业选择**：支持8个不同专业方向
- **选课系统**：合理规划每学期课程
- **主游戏阶段**：每周进行活动安排，提升各项属性
- **考试系统**：期中考试和期末考试
- **夏令营申请**：参与顶尖院校夏令营
- **预推免阶段**：冲刺最终保研名额

### 核心属性
- **GPA**：绩点，保研核心指标
- **科研/项目**：科研背景，论文发表等
- **竞赛**：各类竞赛奖项
- **英语**：英语水平，六级成绩等
- **心态**：心理状态，影响各项活动效果
- **体力**：精力值，限制每周活动次数

### 特色系统
- **随机事件**：丰富多样的校园随机事件，影响游戏进程
- **导师系统**：联系导师，建立关系，获取推荐信
- **申请系统**：申请多所院校，参加面试
- **简历系统**：积累科研和竞赛经历，打造完美简历
- **社交系统**：与同学和学长建立关系
- **商店系统**：购买道具提升效率

## 🛠️ 技术栈

- **框架**：React 18
- **语言**：TypeScript
- **构建工具**：Vite
- **样式**：Tailwind CSS + clsx + twMerge
- **图标**：Lucide React

## 🎮 游戏玩法

1. **开始游戏**：输入你的名字，开始你的保研之旅
2. **高考**：通过模拟高考获得分数
3. **选择大学**：根据高考分数选择合适的大学
4. **选择专业**：从8个专业中选择一个方向
5. **每学期规划**：
   - 选择课程
   - 每周安排活动（学习、科研、竞赛、休息等）
   - 应对随机事件
   - 管理心态和体力
6. **考试准备**：期中考试和期末考试前做好准备
7. **夏令营申请**：大三下学期申请顶尖院校夏令营
8. **预推免申请**：大四上学期进行预推免申请
9. **获得结果**：根据你的表现获得最终保研结果
